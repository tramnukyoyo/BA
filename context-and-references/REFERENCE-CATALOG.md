## How to Add a New Reference

1. **Create folder:**
   ```
   context-and-references/[TYPE]/[colleague-name]/
   ```
   - TYPE: exposés, bachelor-theses, thesis-proposals, master-theses
   - colleague-name: Use underscores (e.g., Anna_Mueller)

2. **Add files:**
   - `document.pdf` (the actual material)
   - `metadata.txt` (basic info)
   - `notes.md` (your analysis)

3. **Update this catalog:**
   - Add to appropriate section table
   - Add to subject category
   - Add to analysis tracking

4. **Commit to git:**
   ```bash
   git add context-and-references/
   git commit -m "ref: Add [colleague-name] — [topic] as reference"
   ```
